Abstract

PECAN advances lightweight, accurate programming language identification using encoder-only neural models. We design, train, and rigorously evaluate efficient transformer encoders that classify programming languages from raw code snippets while maintaining strong generalization across diverse, modern codebases.

We address limitations of heuristic tools like GuessLang and GitHub Linguist by building a unified experimental framework that compares encoder-only model families for accuracy, efficiency, and robustness on large-scale datasets.

Motivation

  • Existing systems struggle with ambiguous snippets and mixed-language repositories.
  • Large-scale codebases demand scalable, efficient classifiers.
  • Lightweight models enable production deployment without sacrificing accuracy.
